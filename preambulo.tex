% fontenc: Define la codificación de fuentes de salida a T1.
%  Efecto: Permite que los caracteres acentuados sean tratados
%  como glifos únicos, mejorando la calidad de la tipografía
%  y el corte de palabras.

% inputenc: Establece la codificación de entrada (UTF-8).
%  Efecto: Permite escribir tildes y eñes directamente
%  en el editor sin caracteres extraños.

% babel: Motor de localización. 
%  Efecto: Traduce automáticamente términos como "Chapter" a
%  "Capítulo" y ajusta las reglas de partición silábica al español.

% lipsum: Generador de texto relleno.
%  Efecto: Inserta bloques de texto "Lorem Ipsum" para realizar 
%  pruebas de diseño sin necesidad de contenido real.

% geometry: Herramienta de control de página.
%  Efecto: Modifica los márgenes y áreas de texto estándar de
%  LaTeX, que suelen ser muy amplios por defecto.

% layout: Herramienta de diagnóstico. 
%  Efecto: Genera un esquema técnico visual con las medidas
%  exactas de la página para verificar el diseño.

% showframe: Herramienta de diagnóstico.
%  Efecto: Dibuja marcos sobre el área de texto y márgenes 
%  para ver visualmente dónde termina cada elemento.

% parskip: Gestor de párrafos.
%  Efecto: Elimina la sangría al inicio de párrafo y añade 
%  un espacio vertical entre ellos (cambia el estilo 
%  "novela" por el estilo "documento técnico").

% setspace: Controlador de interlineado.
%  Efecto: Permite aplicar configuraciones como 
%  \onehalfspacing (1.5), eliminando el espaciado simple por defecto.

% libertine: Familia tipográfica (clásica).
%  Efecto: Cambia la fuente estándar (Computer Modern)
%  por una con estilo editorial profesional.

% libertinus: Familia tipográfica (moderna).
%  Efecto: Proporciona una fuente refinada y 
%  equilibrada, optimizada para lectura técnica.

% libertinust1math: Soporte matemático para Libertinus.
%  Efecto: Asegura que las fórmulas matemáticas usen la 
%  misma familia de fuentes que el texto, evitando la 
%  inconsistencia visual.

% xcolor: Gestión de color. 
%  Efecto: Permite usar nombres de colores (red, blue) 
%  y definir colores personalizados mediante \definecolor.

% graphicx: Interfaz para gráficos.
%  Efecto: Habilita el comando \includegraphics, 
%  esencial para insertar imágenes en formatos estándar.

% tikz: Lenguaje de dibujo vectorial.
%  Efecto: Permite crear diagramas y figuras complejas 
%  mediante código dentro del documento.

% array: Extensión para tablas.
%  Efecto: Añade flexibilidad al entorno tabular, 
%  permitiendo definir tipos de columna más avanzados.

% float: Control de flotantes.
%  Efecto: Permite usar el parámetro [H] para forzar la posición 
%  exacta de una figura, evitando que LaTeX la mueva libremente.

% multicol: Gestor de maquetación.
%  Efecto: Permite dividir secciones del documento en múltiples 
%  columnas, rompiendo la estructura de columna única.

% soul: Decoración de texto.
%  Efecto: Permite realizar subrayados, tachados y resaltados 
%  que se ajustan correctamente a la línea base del texto.

% icomma: Adaptador de coma decimal.
%  Efecto: Elimina el espacio que LaTeX añade por defecto 
%  tras una coma en modo matemático, ideal para el formato español.

% calc: Calculadora.
%  Efecto: Permite realizar sumas y restas con unidades 
%  (cm, pt) directamente en los argumentos de los comandos.

% mdframed: Enmarcador.
%  Efecto: Permite encerrar párrafos en cajas con bordes 
%  personalizados, facilitando la creación de avisos destacados.

% tcolorbox: Motor de cajas decorativas.
%  Efecto: Permite crear cuadros de alto diseño, 
%  con colores de fondo, esquinas redondeadas y títulos.

% tabularx: Tablas de ancho automático.
%  Efecto: Permite el uso de la columna X, que ajusta 
%  automáticamente su ancho para llenar el ancho total del texto.

% booktabs: Reglas de tablas.
%  Efecto: Mejora la calidad visual de las líneas, 
%  desaconsejando (o evitando) el uso de líneas 
%  verticales en tablas.

% caption: Gestor de etiquetas.
%  Efecto: Permite controlar el estilo de los pies 
%  de figura (fuente, negrita, espacio) de forma centralizada.

% subcaption: Gestor de subfiguras.
%  Efecto: Permite anidar imágenes y etiquetas individuales
%   bajo una misma figura principal.

% microtype: Optimizador tipográfico.
%  Efecto: Ajusta el interletraje y el borde derecho 
%  del texto, eliminando espacios desiguales entre palabras.

% enumitem: Personalizador de listas.
%  Efecto: Permite compactar el espacio entre elementos 
%  de las listas (noitemsep), algo que por defecto en 
%  LaTeX suele ser muy disperso.

% appendix: Gestión de anexos.
%  Efecto: Permite cambiar el nombre de los títulos 
%  de secciones a "Anexo" automáticamente al final del documento.

% minted: Resaltado de sintaxis.
%  Efecto: Utiliza una librería externa (Python) para colorear 
%  código fuente con gran precisión visual.

% csquotes: Comillas avanzadas.
%  Efecto: Maneja automáticamente el estilo de las comillas 
%  según el idioma, siendo más inteligente que el teclado.

% biblatex: Motor de bibliografía.
%  Efecto: Sustituye el antiguo sistema de bibliografía 
%  por uno mucho más potente y flexible (estilo IEEE).

% hyperref: Gestión de hipervínculos.
%  Efecto: Convierte todas las referencias, 
%  citas y URLs en enlaces clicables y coloreados.

% bookmark: Gestión de marcadores.
%  Efecto: Crea un índice interactivo en el panel lateral 
%  del lector de PDF, facilitando la navegación.

% cleveref: Referencias inteligentes.
%  Efecto: Automatiza el nombre de la referencia 
%  (ej: cambia "Figura 1" por "Figura 1" automáticamente), 
%  simplificando la escritura de textos.



% siunitx: Gestor de unidades y números.
%  Efecto: Asegura el formato correcto de unidades SI y 
%  decimales, aplicando la coma decimal española  automáticamente 
%  y evitando espacios erróneos en medidas.

% pdflscape: Gestor de orientación de página.
%  Efecto: Permite rotar páginas individuales a formato horizontal mediante el 
%  entorno \begin{landscape}, ideal para tablas anchas o esquemas complejos.

% pdfpages: Gestor de inserción de archivos externos. 
%  Efecto: Permite integrar archivos PDF completos (como 
%  certificados o fichas técnicas) como páginas adicionales
%  de tu propio documento.

%  que tanto el texto como las fórmulas matemáticas compartan la misma

% Configuración de tcolorbox (estilo 'minimalista'):
%  Se ha creado un estilo propio con 'colback=gray!5' y 'sharp corners'
%  para lograr un aspecto moderno, limpio y que no distraiga la atención 
%  del contenido principal del documento.

% Configuración de captions:
%  Se ajusta a 'footnotesize' y 'labelfont=bf' para que las etiquetas de 
%  figuras y tablas no pesen visualmente más que el propio texto del documento.

% Configuración de minted:
%  Se utiliza el estilo 'tango' para resaltar código. Se ha configurado 
%  'breaklines' y 'bgcolor=gray!5' para que el código sea legible incluso 
%  si las líneas son muy largas, evitando que se corten o se salgan del margen.

% Configuración de hyperref:
%  Se activan los enlaces (colorlinks=true) y se asignan colores específicos
%  (azul, rojo, cian) para facilitar la navegación del lector sin que los 
%  enlaces se vean demasiado intrusivos.

% Configuración de siunitx:
%  Se configura el 'output-decimal-marker' como ',' para forzar la notación 
%  decimal española y se usa 'detect-all' para que los números mantengan 
%  siempre la misma fuente que el texto circundante.

% Configuración de enumitem:
%  Se utiliza 'noitemsep' de forma global para compactar las listas,

%  al no dejar huecos excesivos entre elementos.
